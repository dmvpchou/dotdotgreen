# 台灣有機、友善與農產品審核文件指南

> 查核日期：2026-07-15（Asia/Taipei）
> 適用對象：點點綠市集主辦方、攤友審核人員、數位代營運團隊與網站內容編輯
> 文件目的：把攤友使用「有機」「有機轉型期」「友善」「產銷履歷」「CAS」「溯源」「無毒」等說法時，需要查什麼文件、公開資料與檢驗，整理成可執行流程
> 重要聲明：本文件是營運與內容治理參考，不是法律意見、政府驗證或檢驗判定。法規、驗證範圍、證書狀態與檢驗方法可能變動，每次初審、續審及發生疑義時都要回官方系統查核；重大案件應詢問主管機關、驗證機構、實驗室或法律專業人士。

相關文件：[點點綠完整研究與營運科技藍圖](./點點綠完整研究與營運科技藍圖.md)｜[市集科技與數位代營運服務研究](./市集科技與數位代營運服務研究.md)

## 1. 先記住的結論

### 1.1 不要把不同制度混成「有認證」

| 對外類別 | 性質 | 能否代表有機 | 市集主要查核物 |
|---|---|---:|---|
| 有機驗證 | 《有機農業促進法》下的驗證 | 是 | 有效證書、範圍、官方系統、標示／展示 |
| 有機轉型期驗證 | 轉型期間的法定驗證 | 不是完整有機，但可依法稱有機轉型期 | 有效證書、範圍、官方系統、不得使用有機標章 |
| 友善環境耕作登錄 | 農業部審認推廣團體管理下，未經有機驗證的友善耕作 | 否 | 當年度證明、有效推廣團體、農民／土地／作物登錄 |
| 產銷履歷（TAP） | 依 TGAP 與追溯制度的第三方驗證 | 否 | 驗證資訊、追溯碼、品項／批次與有效期 |
| 優良農產品（CAS） | 特定國產農產品生產、加工、分裝及流通的驗證 | 否 | 標章編號、產品、場家、有效期與官方查詢 |
| 溯源農糧產品 QR | 生產者與產品資訊登錄、可追溯 | 否 | QR／追溯編號、登錄者、品項、狀態 |
| 綠色保育標章 | 生態保育取向的專門查證與標章 | 否 | 證書／使用範圍、產品、有效期與核准使用方式 |
| 「無毒／無農藥／自然農法」自述 | 可能是耕作主張、私人制度或地方品牌 | 否；單靠文字無法查驗 | 追問制度、發證者、範圍、紀錄及適切檢驗，不得包裝成官方認證 |
| 單張農藥檢驗報告 | 特定樣品、日期、方法與檢測項目的結果 | 否 | 實驗室、範圍、樣品／批次、日期、方法、結果 |

### 1.2 市集能做的是「審核」，不是另創官方認證

網站與現場建議使用：

- 「市集文件已核驗」。
- 「已於農業部系統查得有效資料」。
- 「本次檢驗報告日期：YYYY-MM-DD」。
- 「市集審核通過，效期至 YYYY-MM-DD」。

避免使用：

- 「點點綠有機認證」。
- 「點點綠無毒認證」。
- 「保證零農藥」。
- 「永久檢驗合格」。
- 讓市集自製徽章看起來像政府有機、TAP 或 CAS 標章。

若未來要建立私人標準，必須公開標準、抽樣方式、申訴與停權流程，並先完成食品標示、廣告、公平交易與責任風險的專業檢視。現階段較安全的定位是**核驗既有證明與紀錄**。

### 1.3 「無毒」不是「有機」的同義詞

本次查核的農業部全國制度中，沒有一個只靠「無毒」二字就能在官方系統核驗的統一全國驗證類別。地方品牌、私人農法或行銷用語可能使用這個詞，但不能因此推論為法定有機驗證。

攤友使用「無毒」時，審核人員應繼續問：

1. 指的是哪一套制度或具體耕作方式？
2. 誰審查、誰發證或誰登錄？
3. 有效期與適用的農地、作物、產品、批次是什麼？
4. 是否有生產紀錄、投入資材或用藥紀錄？
5. 檢驗的是哪一個樣品、哪些項目、哪一天？

花蓮過去曾有地方性的「無毒農業」品牌發展，這更說明「無毒」可能指特定地方或組織制度，而非看到兩字就等於全國有機驗證。歷史背景可參考農業部農業知識入口網：[花蓮無毒農業](https://kmweb.moa.gov.tw/knowledge_view.php?id=499)。

## 2. 市集共通的最低收件包

以下是市集自己的風險管理需求，不代表所有項目都是每位攤友的法定申請文件。

### 2.1 所有農產／食品攤友

| 文件／資料 | 要確認的內容 | 為什麼需要 |
|---|---|---|
| 申請人與攤位負責人 | 姓名／組織、聯絡、地址、緊急聯絡、統編或身分識別方式 | 確認責任主體與事件聯絡 |
| 品牌與完整品項表 | 生鮮、加工、即食、飲品、非食品分開 | 審核不能只審品牌、不審實際販售物 |
| 產品來源聲明 | 自產、共同運銷、代售、批發購入、委託加工 | 決定要看農地、進貨、授權或加工證明 |
| 產地與供應者 | 農場／產地、供應者、可追溯方式 | 發生問題時追到來源 |
| 現行標示照片 | 包裝正反面、散裝告示、標章與 QR | 比對實際對外說法和文件 |
| 合規資料 | 食品業者登錄、商業／攤販許可、保險等「依法或場地方要求時」提供 | 不同業態與規模適用不同，不可一概而論 |
| 市集規約與聲明 | 資料真實、異動通知、抽查、下架、回收、申訴、照片／文案授權 | 建立處理權責與程序 |

食藥署最新 [食品業者登錄 Q&A](https://www.fda.gov.tw/tc/includes/GetFile.ashx?id=f639148063925543656) 說明，是否需要登錄取決於公告指定的業別與規模，並包括部分經登記或地方許可的攤位／攤販。市集不宜只問「有沒有」，而應依攤友的製造、餐飲、輸入、販售業態與實際規模判斷；不確定時請攤友向所在地衛生局或食藥署確認。

### 2.2 非本人到場或代售

另收：

- 生產者／證書持有人出具的代售或供貨授權。
- 進貨單、出貨單或可連回批次的交易文件。
- 證明實際販售品項在證書、登錄或檢驗範圍內的資料。
- 有機散裝產品依法應展示的證書影本及品名、原產地告示。
- 委託加工品的委託關係、原料來源與成品證明。

「我替有機農友賣」不能自動讓攤位所有產品都變成有機；必須逐項、逐來源查核，並避免有機與非有機產品混放造成誤認。

## 3. 各類資格的最低審核標準

### 3.1 有機驗證農產品

#### 法律重點

依 [《有機農業促進法》](https://law.moa.gov.tw/LawContent.aspx?id=GL000940)：

- 有機農業不施用化學肥料及化學農藥，也不使用基因改造生物及其產品。
- 農產品的生產、加工、分裝及流通過程經驗證合格，才能以「有機」名義販賣、標示、展示或廣告。
- 包裝有機產品需標示品名、原料、經營者、原產地、驗證機構與驗證證書字號等法定事項。
- 散裝有機產品在陳列販售處需展示品名、原產地及有機驗證證書影本。
- 主管機關可進入相關場所檢查、抽樣檢驗或要求紀錄。

#### 市集最低收件

1. 有效的有機農產品驗證證書影本。
2. 證書持有人、驗證機構、證書字號與有效期間。
3. 驗證範圍或產品清單，確認涵蓋實際販售品項及相關生產／加工／分裝／流通範圍。
4. 包裝正反面照片；若為散裝，提供現場告示牌樣式及證書影本。
5. 非證書持有人自行販售時，另附供貨／授權、進貨及批次追溯文件。
6. 審核人員在 [臺灣有機農業資訊網](https://epv.afa.gov.tw/Home/OrganicResult) 的查核結果、日期與連結／截圖。

#### 現場必查

- 品項名稱與證書範圍一致。
- 證書仍在有效期，且未顯示終止或暫停。
- 標章、驗證機構與證號和包裝相符。
- 散裝品的品名、原產地、證書影本可見。
- 有機與非有機品項清楚區隔，價格牌與文案不造成整攤都是有機的誤認。

#### 是否要再交農藥檢驗？

有效有機驗證應是主要資格依據。驗證流程本身包含文件審查、現場查驗及依風險進行的抽樣／檢驗；官方也會進行市場抽驗。市集可因客訴、異常、風險品項或抽查計畫要求檢驗，但**不宜自行宣稱法規要求每一批或每月都要附一張檢驗報告**。

農友若正在申請有機驗證，驗證機構會要求比市集更多的內部資料，例如身分／場地、地理資料、生產計畫、作業與投入資材、採購庫存、生產銷售、環境、歷次驗證及客訴紀錄。可參考官方 [如何加入有機驗證](https://epv.afa.gov.tw/Home/ApplyOrganic)。市集看到有效證書後，原則上不需把整套驗證申請檔案再收一次。

### 3.2 有機轉型期農產品

最低收件與有機大致相同，但必須明確區分：

- 證書與官方系統顯示為「有機轉型期」。
- 品名與告示使用完整的「有機轉型期」文字。
- 不得縮寫或設計成讓消費者以為已是有機。
- 依 [有機農產品及有機轉型期農產品標示及標章管理辦法](https://law.moa.gov.tw/LawContent.aspx?id=GL001036)，有機轉型期農產品不得使用有機農產品標章。

網站標籤建議直接顯示「有機轉型期驗證」，不要只顯示「有機」。

### 3.3 友善環境耕作

#### 制度重點

[友善環境耕作推廣團體審認要點](https://law.moa.gov.tw/LawContent.aspx?id=GL000841) 規定：

- 推廣農法需維護水土、生態與生物多樣性，農業生產過程不使用合成化學物質、基因改造生物及其產品。
- 推廣團體需有耕作基準、農民稽核管理規範及農民名冊。
- 團體審認有效期為三年，每次展延最長三年。
- 團體應在指定系統維護農民資料，並可依農民需要開立**當年度**友善環境耕作證明，內容包括農民、地址、土地地段地號、面積、作物及登錄日期。

#### 市集最低收件

1. 當年度友善環境耕作證明文件。
2. 農民姓名、推廣團體、土地、面積、作物與登錄日期。
3. 推廣團體目前仍在農業部審認有效期間。
4. 實際販售作物與登錄作物相符。
5. 委託加工或代售時的來源、授權與批次資料。
6. 在 [友善環境耕作業者／團體查詢](https://epv.afa.gov.tw/Home/FriendlyIndustryQuery) 的查核日期與結果。

#### 對外說法

可以寫：

- 「農業部審認推廣團體登錄之友善環境耕作者」。
- 「友善環境耕作登錄，推廣團體：○○」。

不要寫：

- 「有機驗證」。
- 「政府保證無毒」。
- 「與有機完全相同」。

友善耕作屬有機農業推廣的一部分，但法律明確把它描述為**未經有機驗證**的友善環境耕作。市集應尊重其農法價值，同時向消費者說清楚制度差異。

### 3.4 產銷履歷農產品（TAP）

#### 制度重點

[產銷履歷農產品驗證管理辦法](https://law.moa.gov.tw/LawContent.aspx?id=FL043477) 與 [產銷履歷農產品驗證基準](https://law.moa.gov.tw/LawContent.aspx?id=GL001310&media=print) 要求經營者依臺灣良好農業規範（TGAP）管理風險、留存產銷作業與批次紀錄、接受驗證，並公開產品名稱、經營者、產地、追溯碼、主要作業、包裝日期、驗證機構與有效期間等資料。

#### 市集最低收件／查核

1. 產銷履歷驗證資訊或證書。
2. 追溯碼及產品上的標章／QR 照片。
3. 品項、產地、經營者、驗證機構與有效期間。
4. 到 [產銷履歷農產品資訊網](https://taft.moa.gov.tw/mp-1.html) 輸入追溯碼，確認能查到且與現物一致。
5. 如為代售，保留供應者、進貨及批次連結。

TAP 是生產風險管理、紀錄與可追溯的驗證制度，不等於有機，也不表示每項農藥都「零檢出」。網站應顯示「產銷履歷驗證」，不要改寫成「無毒認證」。

### 3.5 CAS 優良農產品

農業部現行「實施優良農產品驗證制度」公告將其定義為：國內特定農產品的生產、加工、分裝及流通符合驗證基準、具完整可追溯產製紀錄、包裝完整標示清楚，並經驗證機構驗證合格。來源：[實施優良農產品驗證制度](https://law.moa.gov.tw/LawContent.aspx?id=GL001309)。

市集最低查核：

1. 實際產品包裝與 CAS 標章編號。
2. 產品名稱、製造／經營場家及有效日期。
3. 到 [CAS 優良農產品管理入口網](https://cas.moa.gov.tw/) 查詢標章編號、產品與狀態。
4. 只把 CAS 用在查得的產品，不把同品牌其他產品一起標示為 CAS。
5. 加工、分裝、委託代工或標示異動時，確認仍在驗證範圍內。

CAS 是優良農產品的驗證，不等於有機、友善或無農藥。部分舊網頁仍會連到已廢止的《優良農產品驗證管理辦法》；2026 年實務查核應以現行《農產品生產及驗證管理法》、各產品驗證基準及 CAS 官方入口網為準。

### 3.6 溯源農糧產品追溯條碼／QR

2026 年現行 [溯源農糧產品追溯條碼管理作業規範](https://law.moa.gov.tw/LawContent.aspx?id=GL001499&media=print) 用於揭露國產農糧產品生產者資訊、區隔國產與進口並管理追溯條碼。它是追溯與責任資訊，不是有機或品質驗證。

市集最低查核：

1. 追溯條碼／QR 與編號。
2. 到 [溯源農糧產品追溯系統](https://qrc.afa.gov.tw/) 掃描或查詢。
3. 確認登錄者、產品名稱及實際販售者關係。
4. 若系統有政府抽驗資訊，可連結顯示，但仍應標明樣品與日期。
5. 不把「可追溯」改寫成「經驗證無農藥」。

舊的「臺灣農產品生產追溯條碼」制度與條碼已由現行溯源農糧產品制度更新；審核時應接受並查核現行系統，不宜沿用舊圖樣作為唯一判斷。

### 3.7 綠色保育標章

綠色保育標章重點是農業生產與棲地／物種保育的連結，並非全品類通用的政府有機驗證。審核時至少確認：

- 證書或查證資料是否仍有效。
- 使用者、產品及使用情境是否在核准範圍。
- 實際販售品與查證產品一致。
- 宣傳品是否經授權使用標章。

[綠色保育標章使用手冊](https://conservation.forest.gov.tw/File.aspx?fno=69386) 指出，標章只能在證書有效期間及通過查證的產品範圍內使用，若放在產品以外的文宣、看板或電子媒體，需取得授權／核可。市集網站轉貼標章前，也要確認授權範圍。

### 3.8 無任何第三方驗證／登錄，但自述友善、無農藥或減藥

這類攤友不一定不能加入市集，但必須獨立成一類，不能混入有機或官方友善登錄。

建議最低收件：

1. 生產者身分與聯絡資料。
2. 土地位置、地段地號或合法使用／耕作關係資料。
3. 作物、面積、產季與預估供應量。
4. 最近一個生產週期的耕作、投入資材與用藥紀錄。
5. 種苗、肥料、農藥或資材的採購／庫存紀錄；主張完全未使用者也要以紀錄呈現。
6. 產品批次或採收日期的追溯方式。
7. 簽署的生產與宣稱真實聲明、同意市集抽查／訪視及異常下架。
8. 依風險要求近期、能對應實際作物與來源的獨立實驗室檢驗報告。
9. 有條件時申請溯源農糧產品 QR，先建立基本可追溯性。

市集還應做：

- 初次加入前訪視農地或由可信合作單位查核。
- 對高風險或大量供應品項進行不預告抽樣。
- 比較申報面積／產季與實際供應量，找出可能混貨。
- 清楚標示「攤友生產方式自述，經市集文件審核」，不要標示「認證」。

一張合格報告無法證明整個生產週期都未使用農藥，也無法排除其他批次或未列入方法的物質；生產紀錄、來源追溯、訪視與抽樣應一起使用。

## 4. 農藥殘留檢驗怎麼看

### 4.1 三種常被混淆的意思

| 報告用語 | 正確理解 | 不能推論 |
|---|---|---|
| 未檢出（ND） | 在該樣品、該方法、該檢測項目及報告極限下未檢出／低於報告界限 | 絕對為零、所有農藥都沒有、其他批次也相同 |
| 檢出但合格 | 檢出值未超過該作物與農藥的現行殘留容許量 | 有機、未使用農藥、無毒 |
| 不合格 | 超過容許量、使用不得檢出的農藥，或其他法規判定不符 | 不等於可只靠重新送同一業者樣品就解除，需依程序追查與處置 |

農藥殘留容許量會滾動修正；食藥署 2026-04-21 才修正第三條附表一。審核報告時應以採樣／判定時適用的現行標準核對，不把過時表格寫死在網站。來源：[食藥署 2026-04-21 修正公告](https://www.fda.gov.tw/tc/newsContent.aspx?cid=3&id=31518)。

### 4.2 市集可接受的完整報告要素

| 欄位 | 審核問題 |
|---|---|
| 實驗室 | 名稱、地址、聯絡、報告編號是否可驗真？ |
| 認證 | 是否具 TAF／ISO/IEC 17025 認證？該作物／基質、方法與檢測項目是否在認證範圍？ |
| 樣品 | 品名、品種、來源、農場、批次／採收日是否能連到攤上產品？ |
| 抽樣 | 誰採樣、在哪裡採、是否封樣？是攤友自行送樣或市場／第三方抽樣？ |
| 時間 | 採樣、收樣、檢驗、報告日期；是否仍能代表目前供應？ |
| 方法 | 方法代碼、儀器／程序、檢測項目清單是否清楚？ |
| 能力 | 定量極限／報告界限、單位及適用範圍是否足以作判定？ |
| 結果 | 每個檢測項目的數值、ND、容許量與判定是否清楚？ |
| 完整性 | 頁碼、簽章、電子驗證、修改註記及附件是否齊全？ |

可在 [TAF ISO/IEC 17025 測試實驗室認證名錄](https://www.taftw.org.tw/directory/scheme/testLab/search/) 查實驗室；不能只看到實驗室宣稱「通過 TAF」就接受，還要看**本次檢驗方法、基質與項目**是否落在認證範圍。

### 4.3 應要求哪種檢驗

食藥署公布的 [食品中殘留農藥檢驗方法清單](https://www.fda.gov.tw/tc/sitelist.aspx?pn=1&scid=174&sid=103) 包含多重殘留分析方法，例如 MOHWP0055.05 的 410 項方法；但「410 項」不是所有可能農藥的同義詞，特定農藥、作物或風險可能需要其他多重或單一方法。

建議做法：

1. 先告知實驗室作物、來源、耕作／用藥風險與審核目的。
2. 請具能力的實驗室建議適用方法與項目，不只用「最便宜幾百項」選報告。
3. 高風險、客訴或初篩異常案件，採獨立抽樣與確認方法。
4. 快速篩檢可做現場風險篩選，但除非方法、實驗室與認證範圍足以支持正式判定，不要用初篩結果替代完整實驗室報告。
5. 對有機案件，若涉及禁用物質疑義，交由驗證機構／主管機關指導處理，不由市集自行宣告撤銷驗證。

農業試驗所介紹的質譜快速技術也仍以 LC-MS/MS、GC-MS/MS 等儀器進行分析；「快速」不等於目視或簡單試紙即可保證。參考：[農藥殘留質譜快速分析](https://www.acri.gov.tw/Item/Detail/%E9%A3%9F%E5%AE%89%E6%8A%8A%E9%97%9C%E6%96%B0%E5%88%A9%E5%99%A8-%E5%9C%8B%E9%9A%9B%E6%9C%80%E9%80%9F%E8%BE%B2%E8%97%A5%E6%AE%98%E7%95%99%E8%B3%AA%E8%AD%9C%E5%BF%AB%E9%80%9F)。

### 4.4 檢驗頻率不要假裝是全國統一法定值

市集可制定自己的風險政策，但應寫成「市集要求」，不要說成所有農友依法都要月驗。

建議分級：

| 風險層級 | 例子 | 建議作法 |
|---|---|---|
| 低 | 有效有機／TAP 等驗證、資料一致、無異常 | 文件與官方系統為主；依年度抽查計畫抽樣 |
| 中 | 友善登錄、自述農法、品項或來源常變 | 續審、紀錄抽查、定期或季節性檢驗 |
| 高 | 葉菜／莓果等密集防治風險、初次加入、供應量異常、曾缺件 | 入場前近期報告、農地訪視、不預告抽樣 |
| 事件 | 客訴、氣味／外觀異常、官方警示、報告疑似造假、來源不符 | 先暫停涉案批次，保存證據，獨立抽樣並通報／諮詢 |

頻率還要依品項、季節、產區、供應量、既往紀錄與抽樣資源調整。建議每年公開「抽樣政策與彙總結果」，不公開不必要的個資或未完成調查的指控。

### 4.5 土壤與灌溉水檢驗何時需要

對已有有效驗證的農友，市集通常不需重做驗證機構已管理的全部環境檢查。對無第三方驗證者，以下情況可要求：

- 農地鄰近污染源、工業區、廢棄物或高風險水源。
- 新開墾／新承租且無歷史資料。
- 作物有重金屬或特定污染累積風險。
- 客訴、主管機關警示或過去檢測異常。
- 私人標準明確將土壤／水質列為必要項目。

土壤、水與成品回答的問題不同；一份水質合格不能替代成品農藥殘留，也不能證明全期耕作方式。

## 5. 加工食品、飲品與餐食攤位

### 5.1 有機原料不等於有機成品

《有機農業促進法》把有機農產品定義為生產、加工、分裝及流通過程符合驗證基準並驗證合格。使用一項有機原料，不會自動讓整個果醬、麵包、飲品或餐食都能標示為有機。

若成品使用「有機」名義，最低要查：

- 成品本身或相關加工／分裝／流通範圍是否在有效有機驗證內。
- 原料、配方、代工與標示是否和證書／核准範圍一致。
- 包裝的驗證機構、證號、經營者與原產地等標示。

若只有部分原料有機，網站應只在有充分依據且不誤導的情況下，描述該具體原料來源；不要替整項成品加上有機徽章。

### 5.2 加工／餐食最低收件

依品項與業態調整：

- 食品業者登錄字號或不適用依據。
- 完整包裝標示照片：品名、內容物／添加物、淨重、廠商、原產地、有效日期、保存、營養、過敏原等適用項目。
- 配方與主要原料來源、進貨憑證。
- 委託代工合約或代工廠資料。
- 冷藏／冷凍、現場加熱、用水、用電與設備需求。
- 產品責任保險（依法、場地方或市集政策要求時）。
- 相關食品、微生物或其他檢驗（依法、品類風險或市集政策要求時）。
- 宣稱有機、TAP、CAS 等時，成品／流程的相應有效證明。

本指南主題是農產資格與農藥檢驗，不能取代完整的食品製造、餐飲衛生、包裝標示與過敏原審查。加工與即食攤位應另建食品安全清單。

## 6. 標示與網站文案規則

### 6.1 可直接採用的公開欄位

每位攤友可顯示：

```text
資格類型：有機驗證／有機轉型期／友善環境耕作登錄／TAP／CAS／溯源 QR／市集文件審核
發證、驗證或推廣單位：
證書／追溯碼：可公開部分或官方查詢連結
涵蓋品項：
有效期限：
市集最近查核日：
檢驗報告：樣品、日期、實驗室、方法與結果連結（如有）
重要限制：該資格不涵蓋的品項或流程
```

### 6.2 建議文案矩陣

| 有的證據 | 可以寫 | 不要寫 |
|---|---|---|
| 有效有機證書且品項相符 | 「有機驗證：○○品項，有效至…」 | 「全農場／全攤永久無農藥」 |
| 有效轉型期證書 | 「有機轉型期驗證」 | 只寫「有機」或使用有機標章 |
| 當年度友善證明 | 「友善環境耕作登錄，推廣團體…」 | 「有機認證」「政府無毒認證」 |
| 有效 TAP | 「產銷履歷驗證，追溯碼…」 | 「有機」「零農藥」 |
| 有效 CAS | 「CAS 優良農產品，產品…」 | 把整個品牌所有品項都標 CAS |
| 有效溯源 QR | 「可查生產者與產品來源」 | 「通過有機／無毒驗證」 |
| 一張檢驗 ND | 「本樣品於某日以某方法檢驗，報告項目未檢出」 | 「絕對零農藥」「以後都安全」 |
| 檢出但低於 MRL | 「依本次報告判定符合殘留容許量」 | 「無農藥」「未檢出」 |
| 僅有攤友聲明 | 「生產方式由攤友自述，市集已收件」 | 「市集認證」「官方認證」 |

食品標示、宣傳或廣告不得不實、誇張或易生誤解；無證據或證據不足也可能被認定有問題。來源：[食藥署食品標示宣傳規範](https://www.fda.gov.tw/TC/siteContent.aspx?sid=12345)、[認定準則](https://www.fda.gov.tw/tc/includes/GetFile.ashx?id=f638537000011808495)。官網、Facebook、攤位牌與口頭銷售說法都應使用相同的證據邏輯。

### 6.3 對目前網站內容的建議

若網站目前以「有機無毒」統稱所有攤友或整個市集，建議後續內容改版時調整為：

> 「匯集有機驗證、友善環境耕作、產銷履歷與可追溯的在地農產。」

或：

> 「市集依不同品項核驗證書、官方登錄、來源紀錄及必要檢驗；各攤友資格以品牌頁標示為準。」

這比籠統的「無毒」更可信，也方便把每一種制度的查核日期與範圍呈現給顧客。

## 7. 審核流程與狀態

### 7.1 標準流程

```text
收件
  ↓
分類：有機／轉型期／友善／TAP／CAS／溯源／自述／加工
  ↓
文件完整性檢查
  ↓
官方系統與證書範圍查核
  ↓
品項、來源、標示與現場說法比對
  ↓
必要時補件、訪視或檢驗
  ↓
具名審核者決定
  ↓
通過／條件通過／缺件／範圍不符／拒絕
  ↓
排班前效期檢查＋定期續審＋風險抽查
```

### 7.2 系統狀態

| 狀態 | 意義 | 能否排班／公開 |
|---|---|---|
| `draft` | 尚未送審 | 否 |
| `submitted` | 已收件待審 | 否 |
| `missing_documents` | 缺件，已通知期限 | 原則否 |
| `pending_verification` | 文件齊，待官方／人工查核 | 否 |
| `conditional` | 低風險缺項，限期完成 | 依具名核准與限制 |
| `approved` | 指定品項與期間通過 | 是，只限核准範圍 |
| `scope_mismatch` | 證明不涵蓋實際品項／流程 | 否，或移除該宣稱後重審 |
| `expired` | 證書／登錄／市集審核到期 | 否，或移除該宣稱後依一般類重審 |
| `suspended` | 事件調查中暫停 | 否 |
| `rejected` | 不符合或逾期未補 | 否，可依規則申訴／重申請 |
| `withdrawn` | 攤友自行撤回 | 否 |

不要只有一個 `certified = true/false`。同一攤友可能有三項有機、兩項一般、一道委託加工品；審核必須落到**產品／品項層級**。

### 7.3 建議效期與提醒

- 法定證書／登錄：以官方有效期為準。
- 友善證明：收當年度文件，並確認推廣團體仍有效。
- 檢驗報告：不自動沿用到下一季或其他作物；依市集風險政策設定代表期間。
- 自述農法的市集審核：建議最長一年，且每一主要產季更新品項與紀錄。
- 提醒：到期前 60、30、7 天；排班產生前再做一次資格檢查。
- 發生產品、產地、代工、供應者、證書或標示變更時，不等到到期，立即重審受影響範圍。

## 8. 異常、客訴與停權

### 8.1 立即暫停涉案批次的情況

- 來源、品項或證書範圍明顯不符。
- 證書、QR 或檢驗報告疑似變造。
- 主管機關、驗證機構或官方系統顯示終止、暫停或下架。
- 農藥殘留、微生物、過敏原、標示或其他食安疑義。
- 攤友拒絕提供追溯資料或阻礙合理查核。
- 以有機、TAP、CAS 等名義銷售未涵蓋產品。

### 8.2 證據保全

- 記錄日期、站點、攤位、品項、批次、數量與當事人。
- 拍攝包裝、標示、價格牌、證書展示與現場配置。
- 保存收據、進貨／供貨資料及對話。
- 若要採樣，使用事先制定的抽樣、封樣、送驗與留樣流程。
- 不在調查完成前對外發布超出證據的指控。
- 涉及法定通報、回收或健康風險時，立即聯絡所在地衛生／農業主管機關及相關驗證機構。

### 8.3 公平程序

- 暫停與最終停權分開。
- 書面告知原因、證據、影響品項與回覆期限。
- 由未直接涉入爭議者複核重大決定。
- 提供補件、改善、申訴或重新申請方式。
- 相同風險使用相同規則，不因關係、贊助或攤位人氣而改變。

### 10.2 無第三方驗證者聲明重點

正式版本應經法務檢視，至少涵蓋：

- 提供資料與宣稱真實完整。
- 不把市集審核描述成政府或第三方認證。
- 品項、產地、投入資材、用藥、供應者與加工方式異動時主動通知。
- 保留足以追到生產／採收／進貨批次的紀錄。
- 同意依市集政策接受文件抽查、農地訪視與產品抽樣。
- 發生疑義時配合暫停販售、調查、下架與依法通報。
- 不合格、造假或拒查時的停權、申訴與重新申請程序。

## 11. 角色分工

| 任務 | 市集主辦方 | 審核／代營運人員 | 攤友 | 驗證／實驗室／主管機關 |
|---|---|---|---|---|
| 制定資格與抽查政策 | 最終負責 | 提案、維護流程 | 被告知／諮詢 | 提供專業意見 |
| 提供真實文件與異動 | 監督 | 收件、提醒 | 最終負責 | 發證／登錄／檢驗 |
| 官方系統查核 | 核准制度 | 執行並留紀錄 | 協助釐清 | 維護官方狀態 |
| 錄取／拒絕／停權 | 最終決定 | 彙整證據，不代決策 | 可申訴 | 依權責處理 |
| 檢驗抽樣 | 核准政策與預算 | 依程序執行／見證 | 配合 | 實驗室分析、主管機關判定 |
| 網站與現場文案 | 核准 | 依證據製作 | 確認自身資料 | 不替市集背書 |
| 食安事件 | 指揮、通報 | 紀錄、聯絡、追蹤 | 追溯與配合 | 法定調查與處置 |

## 12. 導入優先順序

### 第 1–2 週：停止混用名詞

- 盤點官網、Facebook、招募表單、攤位牌的「有機／友善／無毒」說法。
- 建立上述八類資格，不再只用「有認證／沒認證」。
- 對每一現行攤友建立品項清單與來源類型。

### 第 3–4 週：完成高風險清查

- 優先查自稱有機、無毒、零農藥者。
- 查證書有效期、範圍與官方系統。
- 找出代售、委託加工與同攤混售。
- 缺件先標記，不在資料未清楚時替其製作宣稱。

### 第 2 個月：建立續審與抽查

- 上線 60／30／7 天到期提醒。
- 建立檢驗報告審核欄位與 TAF 範圍查核。
- 制定不預告抽樣、農地訪視與異常處理 SOP。
- 與一至兩家合適實驗室確認品項、方法、費用、採樣及報告格式。

### 第 3 個月：公開透明資訊

- 品牌頁顯示資格類型、範圍、有效期與查核日。
- 說明「市集審核不等於政府驗證」。
- 發布檢驗與抽查政策，結果以具背景的彙總方式呈現。
- 將官網首頁籠統的「有機無毒」改為精確分類與查核說明。

## 13. 仍需向專業單位確認的事項

1. 市集實際販售型態下，各類攤友的食品業者登錄、商業／攤販許可與保險適用範圍。
2. 市集若公開證書影本、查詢截圖與標章圖樣，需要哪些個資遮蔽與授權。
3. 有機產品由非證書持有人代售、分裝或組合銷售時，各流程的驗證範圍與標示要求。
4. 部分有機原料的加工食品，可使用哪些不誤導的文字及需具備何種驗證。
5. 不同作物與季節最合適的農藥殘留、重金屬、微生物或其他檢驗組合。
6. 抽樣、封樣、複驗、下架、回收與對外說明的法律與證據流程。
7. 市集自訂「文件已核驗」徽章的圖樣與文字是否可能讓消費者誤認為官方驗證。

## 15. 文件維護規則

- 每季檢查一次主要法規、官方入口與系統網址；法規修正、主管機關公告或重大食安事件後立即更新。
- 每次更新保留查核日期、修改者、來源與變更摘要。
- 不把法規附件中的農藥容許量或實驗室名單複製成永久靜態資料；網站連回官方現行查詢。
- 市集內部政策要標明「市集要求」，不可寫成「法律一律要求」。
- 攤友證書、土地、供貨、檢驗與申訴資料屬敏感營運／個人資料，不應提交到公開 repo。
- 網站公開前，由營運、內容與至少一位熟悉農業／食品法規者共同複核。
